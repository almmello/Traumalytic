# Traumalytic

## Sobre o Projeto
Traumalytic é uma aplicação desenvolvida para facilitar a análise de dados relacionados a traumas psicológicos. Este projeto se concentra em processar e analisar dados coletados através de dois instrumentos principais: o Inventário de Cognições Pós-Traumáticas (PTCI) e a Posttraumatic Stress Disorder Checklist for DSM-5 (PCL-5).

## Objetivos
- **Análise Descritiva**: Inclui medidas de tendência central, dispersão e exame das propriedades da distribuição para variáveis sociodemográficas numéricas e frequência para variáveis categóricas de interesse.
- **Análise Inferencial**: Exploração do padrão de associação entre variáveis através do coeficiente de correlação de Pearson e testes não-paramétricos para distribuições não normais.

## Instrumentos de Avaliação
### PTCI
- Avalia o conteúdo dos pensamentos e crenças após exposição a eventos estressores potencialmente traumáticos.
- Compreende 36 itens, respondidos em uma escala de 1 (Discordo totalmente) a 7 (Concordo totalmente).

### PCL-5
- Checklist para avaliar o TEPT, com 20 itens pontuados de 0 (De modo nenhum) a 4 (Extremamente).

## Clusters de Análise
- **PTCI**: Dividido em 3 clusters: cognições negativas sobre o self, sobre o mundo e sobre autorresponsabilização.
- **PCL-5**: Dividido em 4 clusters: Intrusões, Evitação, Cognições e Humor, Reatividade.

## Ferramentas de Análise
- Análises realizadas usando Python, incorporadas em uma aplicação Streamlit para fácil acesso e manipulação dos dados.
